# CO2_MPP

**MPP from 1st principles**

A project exploring Maximal Power Point (MPP) concepts from first principles with applications to CO2 systems.

## About

This repository contains Jupyter notebooks demonstrating theoretical foundations and practical implementations of MPP tracking and optimization algorithms. The project builds understanding from basic principles up through complete implementations.

## Features

- 📓 Jupyter Notebook-based learning and exploration
- 🔬 First-principles derivations
- 🎯 Practical MPP implementations
- 📊 Visualization and analysis

## Getting Started

### Requirements

- Python 3.7+
- Jupyter Notebook
- NumPy, SciPy, Matplotlib (typical scientific Python stack)

### Installation

```bash
# Clone the repository
git clone https://github.com/eskoantg/CO2_MPP.git
cd CO2_MPP

# Install dependencies (if requirements.txt exists)
pip install -r requirements.txt

# Launch Jupyter
jupyter notebook
```

## Project Structure

- Notebooks are organized from foundational theory to advanced applications
- Each notebook builds on concepts from previous sections
